Property Description
font-size Specifies the size of the font for a block of text

分類

  • 数値で指定
    • 絶対値系：mm、cm、in、pt、pc
    • 相対系：em、ex、px、rem
  • ％で指定
  • キーワードで指定

よく迷う話（px か em か）

アクセシビリティを考慮して、相対指定が一般的 pxが分かりやすいので使いたい、けど、IEだと固定化されるのでemを使う（ズームは大丈夫だが、文字のサイズ機能がダメ） emは親要素に対しての相対指定なので、同じ要素が入れ子になると、福利的に適用されてしまう →要素の構造によって適用が難しくなる 　→CSS3では、ルートの要素に対する相対指定rem（root em）が使えるので、こちらの使用を検討してみる。
